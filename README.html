Read this before using this template!

:
  • Make sure to update the config.js with your information and customization preferences.
  • Each folder corresponds to a weekly assignment.
  • Most HTML you'll edit for each assignment will be inside the <section class="blog"> tag.
  • If using VScode, install the "Live Preview" extension and use the "Go Live" button at the bottom right.
  • VScode also has text-wrap, which may be helpful for HTML. Toggle it with alt + Z.

The rest is HTML boilerplate. Play around and see how each element corresponds to what appears on the website. It's really not that much more complicated than google docs!
